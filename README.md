# Project Name
**Crack Me**
## Technologies Used
- Git
- HTML
- CSS
- JavaScript
- DOM Maniplualtion
## Description
Crack Me is a cyber security and cryptography themed game. The player cracks hidden security terms, either playing solo or against computer . There are two round types: Scramble and Cipher(Caesar-cipher).

## User Stories
- As a user, I want to be able to chose between single player (solo game) or 1 vs computer player.
- As a user, I want to be able to have a start buttuon, so that I can start the game.
- As a user, I want to be able to have a quit button, so that I can quit the game anytime I want.
- As a user, I want to be able to click on the correct letter one at a time using on screen letter keyboard.
- As a user, I want to be able to have an enter button, so that I can submit my letter guessed.
- As a user, I want to be able to have a timer, so that I can crack the word faster under pressure.
- As a user, I want to be able to have a hint button, so that I see the difinitionof the word.


- As a user, I want to be able to have words shown as scrambled, so that I can crack them by placing the letters in the right place.
- As a user, I want to be able to words shown as caeser cipher word and the shift for it, so that I can crack them by decoding the word.

**Single Player**
- As a user, in a solo game I want to keep my high score "best streak".
- As a user, in a solo game I want to be able to lose one life after I gueesed 3 letters incorrectly.
- As a user, in a solo game I want to be able to see a message "cracked" if win, the difinition of the word, and my score.
- As a user, in a solo game I want to be able to see a message revealing the word if lose and its difinition, and my score.

**vs Computer**
- As a user, I want to be able to chose diffuclilty level in 1 vs compuetr mode.
- As a user, I want to be able to win when I am the first to crack 3 words.
- As a user, I want to be able to continue guessing if i guessed corrctly.
- As a user, I want the computer to take over when I guess wrong.
- As a user, I want the computer to take over when my timer ends.
- As a user, I want to take over when the computer guess wrong.
- As a user, I want to be able to be locked and stoped from gueesing after 3 incorrect guesses in a row.
